## Component Output Examples

### Basic Usage

**Code:**

```tsx
<RefineKbd keys={['command', 'K']} />
```

**Output:**

```
в”Ңв”Җв”Җв”Җв”җ   в”Ңв”Җв”Җв”Җв”җ

в””в”Җв”Җв”Җв”ҳ   в””в”Җв”Җв”Җв”ҳ
```
